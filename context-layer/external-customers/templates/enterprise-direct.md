# Security Service Template: Enterprise Direct

## What We Deliver

Full enterprise security package for direct logistics customers on enterprise plans. Includes dedicated security review, compliance support, and priority incident response.

## Security Controls

- **Application Security**: Full SAST/DAST coverage on customer-facing services, dependency scanning, container scanning
- **Data Protection**: Customer-dedicated encryption keys (KMS CMK), data residency controls, backup encryption
- **Access Control**: SSO integration supported, IP allowlisting available, API credentials with custom scoping
- **Network**: WAF protection, DDoS mitigation, VPC isolation for dedicated deployments (if applicable)

## Monitoring Level

- **Tier**: Enhanced
- Dedicated CrowdStrike host group for customer workloads (if dedicated infrastructure)
- Real-time alerting on customer-affecting security events
- Proactive threat intelligence relevant to customer's industry vertical

## Compliance & Reporting

- SOC 2 Type II report provided annually
- PCI-DSS AOC provided (if in scope)
- Quarterly security posture summary available
- Custom compliance questionnaire response (SLA: 5 business days)
- GDPR data processing agreement and DPO contact

## Incident Response

- **SLA**: Acknowledgment within 1 hour for customer-affecting incidents
- **Communication**: Dedicated security contact, Slack Connect channel (optional)
- **Escalation**: Direct to Security Lead, copied to Customer Success Manager
- **Postmortem**: Customer-facing postmortem for P0/P1 incidents within 5 business days

## Pen Testing

- Annual external pen test of customer-facing services (results summary shared)
- Customer-initiated pen testing welcome with 14 days notice and scoping
- Bug bounty program participation details provided on request
