# Security Service Template: SMB

## What We Deliver

Baseline security posture for small-to-medium logistics customers on standard plans. Security delivered through shared platform controls — no dedicated infrastructure.

## Security Controls

- **Application Security**: Covered by platform-wide SAST/SCA/container scanning
- **Data Protection**: Shared encryption (AWS KMS), TLS in transit, standard backup
- **Access Control**: API key authentication, standard rate limits
- **Network**: Shared WAF and DDoS protection via platform infrastructure

## Monitoring Level

- **Tier**: Standard (shared)
- Covered by platform-wide CrowdStrike and SIEM monitoring
- No customer-specific alerting — covered by platform aggregate monitoring

## Compliance & Reporting

- SOC 2 Type II report available on request
- Standard data processing terms in service agreement
- Compliance questionnaire response (SLA: 10 business days)

## Incident Response

- **SLA**: Acknowledgment within 8 business hours for customer-affecting incidents
- **Communication**: Via standard support channels
- **Escalation**: Through Customer Success team

## Pen Testing

- Annual pen test covers shared platform (results summary available on request)
- Customer-initiated pen testing requires 30 days notice, approval, and scoping
